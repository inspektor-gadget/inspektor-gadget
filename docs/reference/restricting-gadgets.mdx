It is possible to limit the gadgets that can be run by using the
`--allowed-gadgets` flag. By default, all gadgets are allowed.

<Tabs groupId="env">
<TabItem value="kubectl-gadget" label="kubectl gadget">
You can specify these options only at deploy time:

```bash
$ kubectl gadget deploy --allowed-gadgets='ghcr.io/inspektor-gadget/gadget@sha256:e13e3859be5ed8cef676a720274480d2748f66fd98cf8d963af6c4c05121526f,ghcr.io/your-repo/gadget@sha256:digest_of_your_gadget'
...
Inspektor Gadget successfully deployed
```
